Automatically adds the meta tag to claim ownership of a site for Nibbler

== Description ==

wpNibbler adds the meta tag in order to claim ownership of a site for http://nibbler.silktide.com.

Simply get the verification code and save it in the options screen and you are done.

== Installation ==

1. Upload `index.php` to the `/wp-content/plugins/wpNibbler` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
* or
1. Search for wpnibbler via Plugins - Add New in the admin area
2. Click on Install
3. Activate plugin

== Frequently Asked Questions ==

= What is nibbler? =

Nibbler is a website analysis tool. It will analyze your website and grade it between 0 and 10.


= Why do I need this plugin? =

Because if you don't use this plugin you have to manually add the verification code to the template. If you change the template you have to do it again. With this plugin the authentication code is automatically added even if you change theme
